#performance 
[k6](https://k6.io/) is a modern, open-source load testing tool.

https://k6.io/
## Installation
https://grafana.com/docs/k6/latest/set-up/install-k6/
Windows:
```bash
winget install k6 --source winget
```
## Writing tests 
Reference: https://grafana.com/docs/k6/latest/get-started/write-your-first-test/
## Running tests
```bash
k6 run --env HOST=http://localhost:8080 test-script.js
```
## Saving output 
```bash
k6 run --out json=results.json test.js
```
